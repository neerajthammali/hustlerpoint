# **App Name**: Hustler Point Hub

## Core Features:

- Article Publishing: Enable admins to create, edit, and publish articles via a simple editor dashboard, storing article data (title, category, author, content link) in Google Sheets/Excel.
- Newsletter Subscription: Allow users to subscribe to the newsletter via an email form that automatically stores their email in Google Sheets/Excel.
- Dynamic Stats Display: Display key statistics on the homepage, pulled dynamically from a Google Sheet/Excel. Display those statistics through animated counters.
- Trending/Editor's Pick Articles: A tool to intelligently suggest Trending Articles and allow manually curating Editor's Pick articles using an LLM, to be displayed in the sidebar.
- Article Display and Sharing: Showcase articles in a blog layout, complete with thumbnails, excerpts, reading time, and share buttons.
- Responsive Design: Ensure the website is fully responsive, adapting to different screen sizes and devices. Provide dark/light mode switch.

## Style Guidelines:

- Background color: Desaturated black (#1A1A1A) for a sleek, modern feel in dark mode.
- Primary color: Vibrant neon blue (#63B5FF) to highlight interactive elements and CTAs.
- Accent color: A slightly darker, analogous blue (#4799E3) will add contrast for hover effects and visual cues.
- Headings: 'Poppins' (sans-serif) in bold for a contemporary, eye-catching headline style. Note: currently only Google Fonts are supported.
- Body: 'Inter' (sans-serif) in regular weight for readability and a clean, modern aesthetic. Note: currently only Google Fonts are supported.
- Minimalist icons, using a single stroke weight, to match the neon blue accent color, communicating key functions in an understated style.
- Clean and modern layout with ample spacing for readability; grid-based structure for articles; clear hierarchy of information; and prominent placement of newsletter signup.
- Subtle fade-in transitions and motion effects using Framer Motion or CSS to enhance user experience without being distracting.
